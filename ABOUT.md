### Using the Eclipse menu label

I gave you a RIFT variable for use in Eclipse Menu labels:

Adding {{isReversedInputs}{ will show true or false

**Example usage:**
```
{ isReversedInputs == true ? "Reversed Inputs ON" : "No Trolls" }
```
this makes it show "Reversed Inputs ON" when its on, otherwise shows "No Trolls"

More in [Eclipse Labels docs](https://github.com/EclipseMenu/EclipseMenu/blob/main/LABELS.md).
